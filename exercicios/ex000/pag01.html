Como funciona a HTML e as CSS? JS?

HTML e CSS, são linguangens porém não são linguanges de programação, então quando você está fazendo um site em HTML e CSS, você não está programando e sim desenvolvendo, então não se diz eu Eu programo em HTML e CSS, você diz Eu desenvolvo em HTML e CSS.

            HTML = HyperText Markup Language = Linguagem de marcação hipertexto
            CSS = Cascading Style Sheets = Folha de estilo em cascata

A linguagem HTML é uma linguagem focada em conteudo, seja texto, imagem, video..., já a linguagem CSS é focada desing, seja cores, sombras, tamanho, posicionamento..., e tem JS, que é uma linguagem de programação focada nas interações, seja menus, animações, popus, validaçôes...

Tags em HTML são < >, por exemplo <h1> titulo </h1>, não pode ter espaço entre o < > ea tag dentro, nesse exemplo o <h1> é a abertura da tag eo </h1> é o fechamento da tag, algumas não possuem fechamento, porem não são muitas, exemplo, <img src="foto,png" alt="exemplo de foto">, alt = paramentro, exemplo de foto = valor, scr = origem.

Para deixar bonito o titulo vamos usar CSS;

            h1{
                font-family:Arial;
                font-size:20pt;
                color:blue;
            }

h1 = seletor, font-family = declaração, toda declaração no CSS tem ; no final, color = porpriedade, blue = valor, resumindo, CSS tem seletores, dentro dos seletores tem declarações, cada declaração é um par de propriedade e valor.

Estrutura basica de documento HTML

            <!DOCTYPE html> = Primeira linha que diz a seu computador que você desenvolvera em HTML.
            <html lang="pt-br"> = Dizendo que seu conteudo é primordialmente em portugues.
            <head> = Onde fica as configurações do site.
                <meta charset="UTF-8"> = Para o site ser compativel com UTF-8, compativel com acentoações.
                <meta http-equiv="X-UA-Compatible" content="IE=edge"> = Ainda não sei oque faz.
                <meta name="viewport" content="width=device-width, initial-scale=1.0"> = Para o site abrir padrão, em uma escala de 100%.
                <title>Document </title> = titulo do site.
            </head>
            <body> = Onde fica o site, o conteudo do site.

Oque é dominio e hospedagem?

            Dominio é o nome que identifica seu site.
            Hospedagem é o local onde seu site vai estar amarzenado.

Não pode existir dois dominio com o mesmo nome, se você deseja ter um dominio é preciso pagar anualmente, com isso você tera direito a usar o nome por um ano, você tambem pode pagar 5 ou 10 anos se desejar.

Hospedagem é um espaço no qual você amarzena seus arquivos, assim como dominios, é um serviço pago, claro possuem opções gratuitas, porem é mais limitado, você paga mensalmente para ter uma hospedagem, tambem pode ser pago anualmente, durante a escolha é preciso ter muito cuidado, veja bem o espaço, memoria, os recursos etc...
